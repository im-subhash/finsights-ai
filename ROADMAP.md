# FinSights AI Roadmap

## Phase 1: Foundation & Architecture

- Frontend Initialization: Build responsive React mobile dashboard.
- Backend & API: Set up Node.js + Express scalable API architecture.
- Database Setup: Configure MongoDB for flexible transaction metadata.

## Phase 2: Zero-Touch Ingestion & AI Processing

- Regex Parsing: Parse user-authorized SMS strings securely and quickly.
- LLM Integration: Use Gemini 1.5 Flash for low-latency extraction.
- Prompt Chaining: Extract structured metadata with semantic merchant disambiguation.

## Phase 3: Advisory Agent & UI Integration

- Goal Mapping: Map spending behavior against user goals.
- Proactive Nudges: Generate actionable advice instead of passive chart-only UX.

## Phase 4: Future Expansion (Post-Hackathon)

- Voice Integration (Phase 1): Natural voice query support.
- Bank Sync (Phase 2): Account Aggregator-based secure sync.
- Gamification (Phase 3): Social savings and leaderboard experiences.
